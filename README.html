fable.GAM

This package provides a tidy R interface to the prophet forecasting procedure using fable. This package makes use of the gam package for R.

Installation

You can install the development version of fable.GAM from GitHub with:

# install.packages("pak")
pak::pak("MauroFalc1/fable.GAM")

Example

library(fable.GAM)
library(fable)
library(tsibble)

deaths <- as_tsibble(USAccDeaths)
fit <- deaths %>%
  model(GAM = GAM(log(value) ~ trend() + season()))

fc <- fit %>% forecast(h = "2 years")
autoplot(fc, deaths)
